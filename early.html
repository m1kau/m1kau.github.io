Early Browsers

WorldWideWeb (Nexus), 1990

    Tim Berners-Lee created the first web browser at CERN in 1990. He first called it WorldWideWeb and later renamed it Nexus. It showed documents and links, and it could even edit pages inside the browser. It only ran on NeXT computers and felt slow compared to today, but it proved the basic idea of using URLs, HTML, and HTTP to share information.

    - First browser and editor

    - Used early web pieces: URL, HTML, HTTP

    - NeXTSTEP computers only

NCSA Mosaic, 1993

    In 1993 the team at NCSA released Mosaic. It ran on Windows, Mac, and Unix, installed easily, and most importantly showed pictures inline with the text. That change made the web feel alive and helped non-experts use it. Mosaic added familiar buttons, bookmarks, and a clear layout, which is why many people called it the web's first “killer app.”

    - Inline images with text

    - Simple toolbar and bookmarks

    - Crossplatform (Windows/Mac/Unix)

Netscape Navigator, 1994

    After leaving NCSA, Marc Andreessen helped start Netscape. Navigator arrived in 1994 and quickly became the leading browser. It felt fast and polished and brought new ideas to the web. Netscape created JavaScript in 1995 so pages could be interactive, and for a while Navigator was the default choice for most users online.

    - Performance and polish

    - Introduced JavaScript (1995)

    - Dominant mid 90s browser
